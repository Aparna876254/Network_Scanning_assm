# Network Scanning Automation

## Overview
This project is a Python-based network scanning toolkit developed as part of a cybersecurity assignment.  
It automates basic network reconnaissance tasks using **Ping, ARP, and Nmap**.

---

## Features
- Ping scanning (single & multiple hosts)
- ARP table scanning (IP–MAC mapping)
- Nmap scanning (multiple scan types)
- Network range scanning (bonus)
- Multi-threaded scanning (bonus)
- CSV export of results (bonus)
- Logging with timestamps (bonus)

---

## Tools & Technologies
- Python 3
- subprocess module
- platform module
- re (regular expressions)
- Nmap
